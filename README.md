# MVP for Selecting Additional Airline Services - Test Task for TECCOD

[View Demo](https://teccod-airline-services-mvp-deploy.vercel.app)

---

## Description

There is a minimum viable product (MVP) for selecting additional airline services such as baggage, meals, seats, Wi-Fi, and priority boarding. The project is built with **Next.js**, **TypeScript** and **Tailwind CSS**.

The application includes:

-   A list of available services with add/remove functionality
-   A summary card calculating the total price of selected services
-   Responsive design with mobile-first approach

---

## Features

-   Add and remove services
-   Calculate total cost dynamically
-   Visual feedback for selected items
-   Responsive UI for mobile, tablet, and desktop
-   Button and icon animations

---

## Technologies

-   Next.js
-   React
-   TypeScript
-   Tailwind CSS

---

## Installation and Running

1. Clone the repository:
   `bash
git clone https://github.com/zhbankova-maria/teccod-airline-services-mvp
cd teccod
`

2. Install dependencies:
   `bash
npm install
`

3. Run the project in development mode:
   `bash
npm run dev
`

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
